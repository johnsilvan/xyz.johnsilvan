My personal site

## Development

1. Clone repo
    ```
    git clone 
    ```
1. Change to it
    ```
    cd xyz.johnsilvan
    ```
1. Install Hugo. Assumes OS is Linux
    ```
    ./task_install_hugo.sh <HUGO_VERSION>
    ```
1. Serve site
    ```
    ./task_serve.sh
    ```
1. Build site
    ```
    ./task_build.sh
    ```
